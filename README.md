# ゲーム のタイトル
# こかんとんスマッシャー
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
こうかとんを動かして上から降りてくる敵を撃って倒すゲーム

## ゲームの実装
###共通基本機能
* こうかとんの画像の配置
* 十字キーでこうかとんを動かす
* スペースキーで攻撃ができる
* 敵の攻撃を当たると残機が減るすべてなくなるとゲームが終了する
### 担当追加機能
* 残機の表示（担当：植森）：敵からの攻撃が当たると画面左下にある３つの残機が減り0になるとゲームが終了する。
