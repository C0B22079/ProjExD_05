# ゲーム のタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公の鳥（こうかとん）がUFO軍団と戦うゲームで，・・・

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
### 担当追加機能
* ビームの種類の追加
### ToDo
- キーによるビームの切り替え実装
### メモ
* 1キー:縦長なビーム、2キー:横長なビーム、0キー:デフォルトのビーム　に切り替わります。
条件として、縦長ビームはスコア50以上、横長ビームはスコア100以上で発動可能です。（デフォルトビームは無条件で発動可能）
main関数のfor event in pg.event.get():の部分を書き換えているのでコンフリクトした場合は修正お願いします。
