# ゲーム のタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
横スクロールのこうかとんシューティングゲーム
* WASDでこうかとんを操作
* SPACEキーで向いている方向にビームを発射
* クリア条件：Bossの撃破

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス(Bird)
* 敵の爆弾に関するクラス(Bomb)
* 主人公キャラクターのビームに関するクラス(Beam)
* 敵が死んだときの爆発クラスに関するクラス(Explosion)
* 敵の生成に関するクラス(Enemy)
* スコア計算に関するクラス(Score)
 
### 担当追加機能
* Bossを生成,動作に関するクラス(Boss)
